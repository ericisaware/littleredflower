# 接龙 / 一条龙 互动 Demo

基于 `接龙一条龙_PRD_V1.md`、`assets/` 设计素材、`历史代码/solitaire_v2` 还原的高保真可点击 H5 Demo。

## 运行

```bash
cd projects/dragon/demo
python3 -m http.server 8777
# 浏览器打开 http://localhost:8777/  （建议移动端/手机模拟器视图，宽 390）
```

直接双击 `index.html` 也可运行（纯静态、无构建步骤）。

## 技术
- 纯 HTML + CSS + 原生 JS（无框架、无依赖），对齐历史 `solitaire_v2` 的视觉语言
- 视觉：天蓝主题 `#4cb4eb` + 红色强调 `#d81800` + 剪纸/纸袋质感 + 节点卡连接环 + 王者金边特效
- 数据口径来源真实素材：
  - `接龙数据1114v2.xlsx`（10 类行为打分、龙等级 17 级、加成系数、王者门槛、锦旗）
  - `接龙游戏文案V3.1.xlsx`（称号/签名文案）

## 文件
| 文件 | 作用 |
|------|------|
| `index.html` | 页面骨架（龙身/榜单/发起/个人中心 4 个 Tab + 弹层容器）|
| `styles.css` | 全部视觉样式 |
| `data.js` | 数据层：行为/等级/加成/文案/榜单/盲盒/节点 mock |
| `app.js` | 交互逻辑：行为参与 → AI 评分动画 → 上龙 → 升级 → 爱心盲盒 |
| `assets/` | 从原始视觉素材压缩的网图引用 |

## 已实现的可点击流程
1. **龙身首页**：龙长度/等级进度条、10 类行为选择网格（未开启行为加锁）、参与人次、热门好事墙、龙身节点卡（王者金边）、节点点赞
2. **10 类好事参与**（每个都可点）：
   - 喂福报/月捐 → 选金额 → 支付 → 评分上龙（≥门槛判王者）
   - 喂书香 → canvas 手写 → AI 评分
   - 喂画卷 → 选灵感词 → AI 生成 → 换一换/确认 → 评分（多步 modal）
   - 喂微笑 → 模拟优图识别 → 评分
   - 遛一遛/静静心 → 滑块 → 评分
   - 早早起 → 时间档位 → 打卡
   - 爱学习 → 3 道题 → 评分
   - 传善举（随身好事）→ 上传 OOTD/故事 → 1 元公益金 + 上龙
3. **AI 评分 → 展示等级**：分数滚动动画 + 普通/优秀/王者档位 + 加成系数计算 → 确认上龙 → 龙增长 + 升级弹窗
4. **爱心盲盒**：每次好事得 1 次，开出公益金/小红花周边/称号徽章（按权重抽取，稀有度区分）
5. **榜单**：本龙日榜 / 4 品类子榜（含首页流量奖励规则）/ 明星榜 + 上周冠军 Banner
6. **发起接龙**：选项目 → 配置可参与好事（实时算加成系数 ×4/×2/×1.4/×1.2/×1）→ 选主题色 → 创建 → 分享海报
7. **个人中心**：龙珠/盲盒统计、称号锦旗、我发起的/我参与的、打开盲盒
8. **规则弹层 / 分享海报弹层**

## 测试
已通过 Playwright 无头浏览器全流程冒烟测试（0 报错）：首页渲染、AI画作评分上龙、捐款 100 元判王者、锁定行为提示、榜单切换、发起加成计算、个人中心切换均正常。
